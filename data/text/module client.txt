### **Partie 1 : Fondamentaux et Opérations de Base**

---

#### **Section 1 : Introduction à la Gestion des Clients**

**1.1. Quel est l'objectif du Module Clients ?**
Le module Clients de Sobrus Pharma est un outil de gestion de la relation client (CRM) qui centralise les informations de la patientèle pour optimiser le service en officine. Il permet de :
- **Personnaliser les interactions** en enregistrant les informations clés des clients.
- **Fidéliser la clientèle** grâce au suivi des achats et aux programmes de fidélité.
- **Optimiser les stocks** en analysant les historiques de consommation.
- **Améliorer la rentabilité** en identifiant les clients les plus importants.
- **Simplifier la gestion administrative** en automatisant certaines tâches comme les rappels de paiement.

**1.2. Quelles sont les fonctionnalités clés du module ?**
- **Gestion des fiches clients :** Créer, modifier et mettre à jour les informations des clients.
- **Gestion des soldes et paiements :** Suivre les crédits, ajuster les soldes et enregistrer les paiements.
- **Programme de fidélité :** Attribuer et gérer les points de fidélité.
- **Communication client :** Envoyer des SMS et des e-mails directement depuis la plateforme.
- **Archivage :** Gérer les fiches des clients inactifs sans les supprimer définitivement.
- **Importation de données :** Ajouter plusieurs clients simultanément via un fichier CSV.

**1.3. Quelle est la différence entre un client régulier, un client passager et une pharmacie ?**
- **Client régulier :** Un client fréquent pour lequel une fiche détaillée est recommandée afin de suivre son historique et ses préférences.
- **Client passager :** Un client occasionnel. La création d'une fiche n'est pas systématique.
- **Pharmacie :** Une autre entité pharmaceutique (confrère), généralement gérée via la section "Confrères".

#### **Section 2 : Premiers Pas avec le Module Clients**

**2.1. Quels sont les prérequis pour utiliser le module ?**
- Vous devez disposer d'un compte utilisateur Sobrus Pharma avec les droits d'accès au module Clients.
- Une connexion internet stable est nécessaire pour la synchronisation des données.

**2.2. Comment accéder au module Clients ?**
1. Connectez-vous à votre compte Sobrus Pharma.
2. Dans la barre de navigation supérieure, cliquez sur l'onglet **Clients**.
3. Sélectionnez **Clients** dans le sous-menu qui apparaît pour accéder au tableau de bord.

**2.3. Comment comprendre l'interface de la liste des clients ?**
- **Tableau des clients :** Chaque ligne affiche les informations essentielles : Nom, Prénom, Email, Téléphone, Organisme de santé, Numéro d’immatriculation et Solde. Un solde débiteur (dû par le client) est en rouge, tandis qu'un solde créditeur (dû au client) est en vert.
- **Boutons d'action principaux :**
    - **Créer :** Pour ajouter un nouveau client.
    - **Importer :** Pour importer une liste de clients via un fichier CSV.
    - **Archives :** Pour accéder à la liste des clients inactifs.
- **Boutons d'action secondaires :**
    - **Rafraîchir :** Pour actualiser la liste et voir les dernières modifications.
    - **Rechercher :** Pour trouver rapidement un client.

**2.4. Comment naviguer entre les clients actifs et archivés ?**
- Pour consulter les clients inactifs, cliquez sur le bouton **Archives**.
- La vue par défaut affiche les clients actifs.
- Utilisez le bouton **Rafraîchir** après avoir archivé ou restauré un client pour mettre à jour l'affichage.

#### **Section 3 : Gérer les Fiches Clients**

**3.1. Comment créer une nouvelle fiche client ?**
1. Depuis le tableau de bord des clients, cliquez sur le bouton **Créer**.
2. Remplissez les champs requis :
    - **Informations générales :** Nom, Prénom, Type de client, Téléphone, Email, et Plafond de crédit.
    - **Partage d’information via My Sobrus :** Cochez les cases pour autoriser le partage du solde, des points de fidélité et de l'historique des transactions.
    - **Info organisme (Tiers payant) :** Renseignez l'organisme de santé, le numéro d’immatriculation et le numéro d’affiliation.
3. Cliquez sur **Sauvegarder**. Pour ajouter un autre client dans la foulée, choisissez **Sauvegarder & Nouveau**.

**3.2. Comment modifier les informations d’un client ?**
1. Localisez le client dans la liste, en utilisant la barre de recherche si nécessaire.
2. Cliquez sur l'icône en forme de **crayon** à droite de la ligne du client, ou sélectionnez le client et choisissez **Modifier** dans le menu "Autres actions".
3. Mettez à jour les informations dans la fiche.
4. Cliquez sur **Sauvegarder**.

**3.3. Comment définir un plafond de crédit pour un client ?**
1. Ouvrez la fiche du client en mode modification.
2. Dans la section "Informations générales", saisissez le montant souhaité dans le champ **Plafond de crédit**.
3. Sauvegardez les modifications. Un plafond de 0 signifie qu'aucun crédit n'est autorisé.

**3.4. Comment renseigner les informations de tiers payant ?**
1. Accédez à la fiche du client en mode modification.
2. Dans la section "Info organisme", sélectionnez l'organisme et saisissez les numéros d’immatriculation et d’affiliation.
3. Sauvegardez les modifications.

**3.5. Comment personnaliser le partage d’informations via My Sobrus ?**
1. Ouvrez la fiche du client en mode modification.
2. Dans la section "Partage d’information", cochez ou décochez les options souhaitées (Solde, Points de fidélité, Historique des transactions).
3. Sauvegardez les modifications.

**3.6. Comment lier une fiche client à un compte My Sobrus ?**
1. Accédez à la fiche du client.
2. Dans le menu "Autres actions", sélectionnez **Lier la fiche client à un compte My Sobrus**.
3. Saisissez le code de vérification que le client vous fournit depuis son application My Sobrus.
4. Confirmez la liaison en cliquant sur **Sauvegarder**.

---

#### **Section 4 : Importer des Clients**

**4.1. Comment préparer un fichier CSV pour l'importation de clients ?**
1. Créez une feuille de calcul avec uniquement deux colonnes : la première pour le **Nom du client**, la seconde pour le **Solde crédit**.
2. Formatez les soldes en utilisant un point comme séparateur décimal (ex : `150.50`) et sans aucun séparateur de milliers.
3. Enregistrez le fichier au format **CSV (délimité par des virgules)**. Le respect strict de ce format est essentiel pour éviter les erreurs.

**4.2. Comment importer des clients avec leur solde crédit ?**
1. Dans le module Clients, cliquez sur le bouton **Importer**.
2. Dans la fenêtre qui s'ouvre, cliquez sur "Sélectionnez ou déposez vos fichiers ici" et choisissez votre fichier CSV.
3. Cliquez sur **Sauvegarder** pour lancer le processus d'importation.

**4.3. Comment importer des contacts associés à un client ?**
L'importation en masse de contacts associés (ex: membres d'une même famille) nécessite une intervention du support technique.
1. Préparez un fichier Excel avec une seule colonne listant les noms des contacts.
2. Identifiez l'ID du client principal (visible dans l'URL de sa fiche client).
3. Envoyez une demande à **support@sobrus.com** avec le fichier Excel et l'ID du client principal en pièce jointe. Seuls les noms des contacts seront importés par cette méthode.

---

#### **Section 5 : Gérer les Statuts et Archives**

**5.1. Comment archiver un client ?**
Vous pouvez archiver un client lorsqu'il devient inactif.
1. Assurez-vous que le solde du client est à zéro. Un client avec un solde non nul ne peut pas être archivé.
2. Ouvrez la fiche du client à archiver.
3. Cliquez sur le bouton **Archiver** (généralement en rouge) situé en haut de la fiche. Le client sera alors déplacé vers la section "Archives".

**5.2. Comment restaurer un client archivé ?**
Si un client archivé redevient actif, vous pouvez restaurer sa fiche.
1. Cliquez sur le bouton **Archives** en haut à droite de la liste des clients pour afficher les clients inactifs.
2. Localisez et ouvrez la fiche du client à restaurer.
3. Cliquez sur le bouton **Restaurer** (généralement en vert) en haut de la fiche. Le client redeviendra actif et réapparaîtra dans la liste principale.

#### **Section 6 : Suivre les Transactions des Clients**

**6.1. Comment consulter l’historique des ventes d’un client ?**
1. Ouvrez la fiche du client concerné.
2. Dans le menu latéral de la fiche, sélectionnez l'onglet **Ventes**.
3. La liste de toutes les ventes s'affiche. Vous pouvez utiliser les filtres pour trier ou rechercher une transaction spécifique. Vous pouvez également créer une nouvelle vente à partir de cet écran en cliquant sur **Créer**.

**6.2. Comment gérer les devis d’un client ?**
1. Accédez à la fiche du client.
2. Dans le menu latéral, sélectionnez l'onglet **Devis**.
3. Vous pouvez y consulter les devis existants, les filtrer, ou en créer un nouveau en cliquant sur **Créer**.

**6.3. Comment créer une facture globale ?**
Une facture globale regroupe plusieurs transactions (ventes, retours) pour un client, ce qui est utile pour les mutuelles ou les entreprises.
1. Accédez à la fiche du client.
2. Cliquez sur l'onglet **Facture globale**.
3. Cliquez sur **Créer** à droite.
4. Sélectionnez les ventes et retours sur vente que vous souhaitez inclure.
5. Ajustez la date de la facture si nécessaire.
6. Cliquez sur **Sauvegarder**.

**6.4. Comment consulter ou imprimer une facture globale ?**
1. Dans l'onglet **Facture globale** de la fiche client, la liste des factures créées s'affiche.
2. Cliquez sur une facture pour la sélectionner.
3. Choisissez **Imprimer** pour générer un document imprimable ou **Supprimer** pour l'effacer. La suppression requiert un code de sécurité.

**6.5. Comment gérer les retours sur vente d'un client ?**
1. Accédez à la fiche du client.
2. Dans le menu latéral, sélectionnez l'onglet **Retours sur ventes**.
3. La liste des retours s'affiche. Pour en créer un nouveau, cliquez sur **Créer**, puis renseignez les informations du produit retourné.

**6.6. Comment suivre les préparations pour un client ?**
Les préparations sont des commandes spécifiques, comme les médicaments composés.
1. Accédez à la fiche du client.
2. Dans le menu latéral, sélectionnez l'onglet **Préparations** pour voir l'historique ou en créer une nouvelle.

**6.7. Comment consulter l'historique des produits pour un client ?**
Pour analyser les habitudes d'achat d'un client, vous pouvez consulter tous les mouvements de produits le concernant.
1. Accédez à la fiche du client.
2. Dans le menu latéral, sélectionnez l'onglet **Historique des produits**.

#### **Section 7 : Gérer les Soldes et Paiements**

**7.1. Comment vérifier le solde d’un client (débiteur/créditeur) ?**
1. Accédez à la fiche du client.
2. Dans le menu latéral, l'encadré du solde affiche le "Total non payé".
   - Un solde en **rouge** indique que le client est débiteur (il doit de l'argent).
   - Un solde en **vert** indique que le client est créditeur (il a un avoir).
   - La mention "Dont un avoir de..." précise le montant de l'avoir en attente d'utilisation.

**7.2. Comment solder un avoir ?**
Pour utiliser un avoir existant dans une transaction :
1. Cliquez sur **"Payer crédit client"**.
2. Sélectionnez le mode de paiement **Avoir**.
3. Indiquez le montant de l’avoir à utiliser.

**7.3. Comment ajuster manuellement le solde d’un client ?**
Cette opération permet de modifier le solde sans générer de transaction de vente.
1. Accédez à la fiche client.
2. Dans le menu latéral, allez à l'onglet **Ajustements de solde clients**.
3. Cliquez sur **Créer**.
4. Renseignez le montant (positif pour ajouter un crédit, négatif pour réduire une dette), le moyen de paiement et une note explicative.
5. Cliquez sur **Sauvegarder**.
**Attention :** Les ajustements de solde n'impactent ni l'arrêté de caisse, ni le chiffre d'affaires.

**7.4. Comment payer des ventes non soldées ?**
Pour régler des factures spécifiques en attente :
1. Sur la fiche client, cliquez sur **Autres actions** puis **Payer ventes non soldées**.
2. Sélectionnez les factures à régler.
3. Modifiez la date ou appliquez une remise si nécessaire.
4. Cliquez sur **Payer**.

**7.5. Comment utiliser la fonction "Payer crédit client" ?**
Cette fonction permet de régler tout ou partie du solde débiteur sans sélectionner de factures spécifiques.
1. Sur la fiche client, cliquez sur le bouton **Payer crédit client**.
2. Indiquez le montant à payer et le mode de paiement.
3. Vous pouvez modifier la date ou appliquer une remise.
4. Cochez "Utiliser le montant à rendre comme avoir" si vous souhaitez que la monnaie rendue soit créditée sur le compte du client.
5. Cliquez sur **Payer**.

**7.6. Comment créer un avoir (paiement en avance) ?**
Pour enregistrer un paiement reçu en avance :
1. Sur la fiche client, allez à l'onglet **Avoirs**.
2. Cliquez sur **Créer**.
3. Renseignez le type d'avoir (ex: "avance"), le montant, le moyen de paiement et une référence.
4. Cliquez sur **Sauvegarder**.

**7.7. Comment appliquer une remise par défaut à un client ?**
1. Sur la fiche client, cliquez sur **Autres actions** puis **Modifier remise par défaut**.
2. Saisissez le pourcentage de remise souhaité pour chaque catégorie de produits.
3. Cliquez sur **Sauvegarder**. La remise s'appliquera automatiquement aux futures ventes de ce client.

#### **Section 8 : Gérer les Points de Fidélité**

**8.1. Comment consulter le solde de points de fidélité d'un client ?**
1. Accédez à la fiche du client.
2. Dans le menu latéral, sélectionnez l'onglet **Ajustements des points de fidélité**. Le solde actuel est affiché en haut de la page.

**8.2. Comment ajouter ou annuler manuellement des points de fidélité ?**
1. Allez dans l'onglet **Ajustements des points de fidélité** de la fiche client.
2. **Pour ajouter des points :** Cliquez sur **Créer**, saisissez le nombre de points et une note, puis sauvegardez.
3. **Pour annuler un ajustement :** Sélectionnez l'ajustement dans la liste, cliquez sur **Annuler** et confirmez avec votre code de sécurité.

**8.3. Comment offrir un cadeau avec les points de fidélité ?**
1. Allez dans l'onglet **Ajustements des points de fidélité** de la fiche client.
2. Cliquez sur l'option **Cadeaux disponibles**.
3. Sélectionnez un cadeau dans la liste proposée (produit ou avoir).
4. Confirmez l'échange pour déduire les points du solde du client.

**8.4. Que se passe-t-il lorsqu’un client utilise ses points pour un cadeau ?**
- Si le cadeau est un **produit**, une vente est créée avec une remise de 100%.
- Si le cadeau est un **avoir**, un paiement est enregistré dans l'onglet **Avoirs** du client, avec une note indiquant son origine ("Avoir transformé via points de fidélité").

**8.5. Comment télécharger un relevé des points de fidélité ?**
1. Allez dans l'onglet **Ajustements des points de fidélité** de la fiche client.
2. Cliquez sur **Relevé et cadeau des points de fidélité**.
3. Choisissez la période souhaitée.
4. Téléchargez le relevé au format PDF.

**8.6. Comment sont définies les conditions d'attribution des points ?**
Les règles, comme le nombre de points attribués par montant dépensé, sont configurées par le pharmacien dans les paramètres généraux de Sobrus Pharma, et non sur la fiche client individuelle.

#### **Section 9 : Communiquer avec les Clients**

**9.1. Comment envoyer un SMS à un client pour un crédit à payer ?**
1. Accédez à la fiche du client concerné.
2. Cliquez sur **Autres actions** en haut de la fiche.
3. Sélectionnez **Envoyer le crédit à payer par SMS**.
4. Un message pré-rempli s'affiche. Vous pouvez le personnaliser si besoin.
5. Cliquez sur **Envoyer**.

**9.2. Comment envoyer un e-mail à un client (ex: facture, devis) ?**
1. Accédez à la transaction que vous souhaitez envoyer (par exemple, une facture depuis l'onglet **Ventes** ou un devis depuis l'onglet **Devis**).
2. Cliquez sur le document pour l'ouvrir ou le sélectionner.
3. Cliquez sur **Autres actions**, puis sélectionnez **Envoyer par E-mail**.
4. Personnalisez l'e-mail si nécessaire et cliquez sur **Envoyer**.

**9.3. Comment consulter l'historique des communications avec un client ?**
1. Accédez à la fiche du client.
2. Dans le menu latéral, sélectionnez :
   - L'onglet **Communication** pour voir l'historique des SMS de recouvrement envoyés.
   - L'onglet **E-mails** pour consulter l'historique des e-mails transactionnels (factures, devis, etc.) envoyés via Sobrus Pharma.

**9.4. Comment personnaliser les modèles de messages (SMS et e-mails) ?**
La personnalisation des modèles de communication se fait dans les **paramètres généraux** de Sobrus Pharma. Cette configuration est généralement gérée par le pharmacien responsable pour assurer une cohérence pour toute l'officine.

#### **Section 10 : Gérer les Contacts Associés**

**10.1. Qu'est-ce qu'un contact associé ?**
Un contact associé est une personne liée à un client principal (par exemple, un membre de la famille, un représentant d'une société). Cela permet de gérer les transactions pour des entités complexes tout en les reliant à un seul compte principal.

**10.2. Comment créer un contact associé pour un client ?**
1. Accédez à la fiche du client principal.
2. Dans le menu **Autres actions**, sélectionnez **Créer un nouveau contact**.
3. Remplissez les champs obligatoires : **Nom**, **Type de relation** (ex: "Conjoint", "Enfant"), et assurez-vous que le champ **Client relié** pointe vers le bon client principal.
4. Cliquez sur **Sauvegarder**.

**10.3. Comment modifier les informations d'un contact associé ?**
1. Accédez à la fiche du client principal.
2. Dans le menu latéral, sélectionnez l'onglet **Contacts**.
3. Repérez le contact à modifier dans la liste et cliquez sur l'icône **Crayon** à droite de son nom.
4. Mettez à jour les informations et cliquez sur **Sauvegarder**.

**10.4. Comment suivre les transactions spécifiques à un contact associé ?**
1. Dans l'onglet **Contacts** de la fiche client principale, cliquez sur le nom du contact concerné.
2. Des sous-onglets apparaissent, vous permettant de consulter les transactions spécifiques à ce contact : **Devis**, **Ventes**, **Bons de commande**, etc.

**10.5. Comment consulter l'historique des modifications ou ajouter des commentaires pour un contact ?**
1. Sélectionnez le contact dans l'onglet **Contacts** de la fiche client principale.
2. Utilisez les sous-onglets pour :
   - **Informations de traçabilité :** Voir qui a créé ou modifié la fiche du contact et quand.
   - **Commentaires :** Ajouter ou consulter des notes spécifiques aux transactions ou interactions de ce contact.

---

#### **Section 11 : Générer des Relevés Clients**

**11.1. Comment télécharger un relevé client détaillé ?**
Un relevé client récapitule toutes les opérations (ventes, paiements, avoirs, retours) sur une période donnée.
1. Accédez à la fiche du client.
2. Cliquez sur le bouton **Relevé client** en haut de la fiche (raccourci : ⇧+P).
3. Choisissez la **période** souhaitée.
4. Sélectionnez le **type de relevé** :
   - **Simplifié :** Détaille les transactions par produit.
   - **Avancé :** Inclut des informations supplémentaires comme les numéros de facture et les remises.
5. Cliquez sur **Télécharger** pour obtenir le relevé au format PDF.

**11.2. Comment obtenir un relevé regroupé par contact ?**
Cette option est utile pour les clients ayant plusieurs contacts associés, afin de distinguer les transactions de chacun.
1. Accédez à la fonction **Relevé client**.
2. Choisissez la période.
3. Avant de télécharger, cochez la case **Regroupés par contact**.
4. Cliquez sur **Télécharger**.

#### **Section 12 : FAQ - Questions Fréquemment Posées**

**12.1. Comment corriger une erreur (ex: mauvais numéro de téléphone) dans une fiche client ?**
1. Accédez à la fiche du client.
2. Cliquez sur l'icône **Crayon** ou sélectionnez **Modifier** dans le menu "Autres actions".
3. Corrigez l'information erronée dans le champ approprié.
4. Cliquez sur **Sauvegarder**.

**12.2. Que faire si je ne parviens pas à archiver un client ?**
L'archivage échoue si le solde du client n'est pas à zéro. Pour résoudre ce problème :
1. Vérifiez le solde du client sur sa fiche.
2. Si le solde est non nul (débiteur ou créditeur), vous devez le ramener à zéro.
   - **Pour un solde débiteur (le client doit de l'argent) :**
     - Si le client a payé, utilisez la fonction **"Payer crédit client"**. Cette action impactera l'arrêté de caisse.
     - Si le client n'a pas payé et que le but est seulement de solder le compte pour l'archiver, utilisez un **ajustement de solde** en négatif. Cette action n'impactera pas la caisse.
   - **Pour un solde créditeur (le client a un avoir) :**
     - Utilisez l'avoir dans une transaction ou effectuez un ajustement de solde.
3. Une fois le solde à zéro, vous pouvez archiver le client en cliquant sur le bouton **Archiver** sur sa fiche.
